GearSave 🚀
💡 Gerencie sua produtividade, finanças automotivas e hábitos em um só lugar!

📌 Sobre o projeto
O GearSave foi criado para ajudar pessoas que vivem rápido demais e acabam se desorganizando. Combinando ferramentas de produtividade, controle financeiro automotivo e gestão de hábitos, ele ajuda a manter o foco, a organização e o controle financeiro dos seus veículos.

🔥 Funcionalidades
✅ Gestor de Produtividade para Programadores – Organize tarefas, use Pomodoro e integre com o GitHub.
✅ Gerenciador de Gastos para Carros e Motos – Registre abastecimentos, manutenções e veja gráficos de análise.
✅ Organizador de Anotações e Código – Bloco de notas + snippets organizados, com modo Quadro Negro.
✅ Assistente Pessoal via Terminal – Responde perguntas, exibe lembretes e até conta piadas!
✅ Controle de Hábitos e Desafios – Defina desafios e acompanhe seu progresso.
✅ Painel de Status do Usuário – Exibe produtividade, humor, tempo de foco e progresso no estilo RPG (skills).

🛠 Tecnologias utilizadas
🔹 Backend: FastAPI + PostgreSQL
🔹 Frontend: React.js + React Native
🔹 Banco de Dados: PostgreSQL
🔹 Autenticação: JWT
